## Demo

https://ts-calculator.herokuapp.com/

## Run Locally

Clone the project

```bash
  https://github.com/yohanessahrul/ts-calculator.git
```

Go to the project directory

```bash
  cd ts-calculator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

  
## Tech Stack

**Client:** React, Redux, Sass
